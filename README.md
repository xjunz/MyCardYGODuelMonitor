# MyCardYGODuelMonitor
萌卡观战助手是一款监视萌卡(MyCard)平台上的YGO竞技对局并实施推送的安卓程序。<br/>
目前支持竞技对局监测、对局双方DP排名显示、自定义推送、直接跳转到YGO Mobile观战、天梯前100决斗者及其对应卡组监测。<br/>
